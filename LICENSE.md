## 🔧 Common Issues and Fixes

**❌ Pet spawner produces non-tradable pets**
→ Ensure you are on v5.2.0 or later. Older versions did not include
  the tradable generation fix. Update the script and re-execute.

**❌ Auto farm stops after a few minutes**
→ The anti-AFK module may have conflicted with a recent Adopt Me update.
  Toggle it off and back on in the GUI, or update to the latest version.

**❌ GUI doesn't appear after execution**
→ Your executor may have incomplete UNC support. Switch to a Level 7
  executor such as Synapse Z, Delta, or Xeno and retry.

**❌ Auto accept isn't triggering on incoming trades**
→ Check that the **adopt me script auto** accept toggle is enabled in the
  Trade tab of the GUI. Also confirm your filter settings aren't excluding
  all offers.

**❌ Script errors with "attempt to index nil" on pet spawn**
→ The pet name list may be outdated after a game update. Update to the
  latest script version where the pet database has been refreshed.

**❌ Mobile GUI is too large / off-screen**
→ Open the GUI settings tab and reduce the UI scale to 75% or 80%.
  This is a known issue on phones with screen resolutions below 1080p.

**❌ Script works on PC but not Android**
→ Use the **spawn pet adopt me script** mobile configuration preset.
  Some PC-specific API calls are not compatible with mobile executor
  environments — the mobile preset handles this automatically.
